# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звёздочкаи (*). Например: *"Пример"*

ЧТобы выделить тест полужирным, необходимо обрамить его двойными (**), Например: **"пример"**
